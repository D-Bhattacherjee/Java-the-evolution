### Phase 1: Absolute Beginner (Syntax & Logic)
*Master the absolute baseline mechanics of code execution flow before optimization matters.*

* **Hello World & Basic I/O**
    * *Problem:* Take a user's name as string input and print a personalized greeting.
    * *Focus:* Input/Output streams.

* **Even or Odd**
    * *Problem:* Take an integer input and determine if it is even or odd.
    * *Focus:* Modulo operator (`%`) and conditional branching (`if/else`).

* **Fahrenheit to Celsius**
    * *Problem:* Create a temperature converter using the formula: $C = (F - 32) \times \frac{5}{9}$.
    * *Focus:* Floating-point arithmetic and data type casting.

* **🚀 Level Up: Leap Year Checker**
    * *Challenge:* Write a program to check whether a given year is a leap year or not.
    * *Why it's tougher:* Introduces nested logical operators (`AND`, `OR`) because a year must be divisible by 4, but not by 100 unless it is also divisible by 400.

* **🚀 Level Up: Multiplication Table**
    * *Challenge:* Given an integer $N$, print its complete multiplication table from 1 to 10.
    * *Why it's tougher:* Introduces loop control variables and tabular formatting.
